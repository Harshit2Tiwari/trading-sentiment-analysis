## Hyperliquid Trader Sentiment Analysis

Exploratory data analysis of 211,224 trades from Hyperliquid 
(May 2023 – May 2025) correlated with the Bitcoin Fear & Greed Index.

### Key Findings
- Greed periods generate 3x more PnL than Fear periods
- Average PnL per trade peaks at $78.69 during Extreme Greed
- ETH earns $323/trade in Greed but loses $184 in Extreme Greed
- Win rate alone does not predict profitability
